# Quick Grading Checks

For each challenge, we provide inputs in `data/` and expected outputs in `expected/`.

### Challenge 1
- Input: `challenge1/data/data1.csv`
- Expected cleaned values: `challenge1/expected/cleaned_values.json`
- Expected stats (mean, median): `challenge1/expected/stats.json`
- Sample stdout: `challenge1/expected/sample_stdout.txt` (informational)

### Challenge 2
- Data points: `challenge2/data/points4.txt` (four 2D points)
- Expected distance matrix: `challenge2/expected/distances4.json`
- Sample stdout: `challenge2/expected/sample_stdout.txt`

### Challenge 3
- Input: `challenge3/data/values.txt`
- Expected cleaned values: `challenge3/expected/cleaned_values.json`
- Sample stdout: `challenge3/expected/sample_stdout.txt`
- Example plot image (for reference): `challenge3/expected/hist_example.png`

### Challenge 4
- Input: `challenge4/data/input.txt`
- Expected top 3 words: `challenge4/expected/top3.txt`
- Full word counts (reference): `challenge4/expected/counts.json`
